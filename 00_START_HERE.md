## 🚀 Quick Start (5 minutes)

```bash
# 1. Configure environment
cp .env.example .env
# Add your API keys

# 2. Install dependencies
cd backend && npm install
cd frontend && npm install

# 3. Start services
npm start              # Backend on :5000
npm run dev            # Frontend on :5173

# 4. Access feature
http://localhost:5173/market
```
