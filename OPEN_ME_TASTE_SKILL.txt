🚀 HOW TO SEE IT LIVE IN YOUR BROWSER

STEP 1: Add Demo to App.jsx
────────────────────────────

Open: frontend/src/App.jsx

Add this import at the top:
  import TasteSkillDemo from './pages/TasteSkillDemo';

Find your page routing (switch or if-else) and add:
  case 'taste-demo':
    return <TasteSkillDemo />;

STEP 2: Add Navigation Button
──────────────────────────────

In your navigation component, add:
  <button onClick={() => setCurrentPage('taste-demo')}>
    🎨 Taste Demo
  </button>

STEP 3: View in Browser
────────────────────────

Start dev server:
  cd frontend
  npm run dev

Click your new "🎨 Taste Demo" button!

🎨 COPY-PASTE EXAMPLES (USE ANYWHERE!)
═══════════════════════════════════════════════════════════════════════════

Import (add to any component):
───────────────────────────────
import { tastePatterns, motionPresets } from '../tasteSkillConfig';
import { motion } from 'framer-motion';

Example 1: Premium Card
───────────────────────
<motion.div
  className={`${tastePatterns.premiumCard.base} ${tastePatterns.premiumCard.dark}`}
  {...motionPresets.scaleIn}
>
  <h3>Soil Health</h3>
  <p className="text-3xl font-bold">85%</p>
</motion.div>

Example 2: Gradient Heading
────────────────────────────
<h1 className={tastePatterns.heading.h1}>
  AgriTech Dashboard
</h1>

Example 3: Premium Button
──────────────────────────
<motion.button
  className={tastePatterns.premiumButton.primary}
  whileHover={{ scale: 1.05 }}
>
  View Analytics
</motion.button>

Example 4: Success Alert
────────────────────────
<div className={tastePatterns.premiumAlert.success}>
  ✓ Optimal NPK Levels - Continue current schedule
</div>
