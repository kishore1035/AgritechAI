🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Issue: Setup script won't run
Solution: Right-click → "Run as administrator"
          OR use PowerShell: .\setup-taste-skill.bat

Issue: Git clone fails
Solution: Install Git from https://git-scm.com
          Restart terminal and try again

Issue: Import errors
Solution: Use relative imports: import { tastePatterns } from '../skills'
          NOT absolute: import { tastePatterns } from 'skills'

Issue: Framer Motion not working
Solution: cd frontend && npm install framer-motion

Issue: Styles not applying
Solution: Ensure Tailwind CSS is properly configured
          Check tailwind.config.js includes src paths
