## 🎬 Animations at a Glance

### Button Component
```

Tap:      Scale 0.95 + y: 0px ↓
Physics:  Spring stiffness 400, damping 25
Feel:     Snappy, responsive feedback
```

### Card Component
```
Load:     Opacity 0→1 + y: 20→0 (fade + lift)
Hover:    y: -4px + shadow increase
Physics:  Spring stiffness 300, damping 30
Feel:     Smooth entrance, depth perception
```

### Modal Component
```
Open:     Backdrop fade (0→1) + Dialog scale (0.95→1)
Close:    Reverse animations smooth
Physics:  AnimatePresence mode="wait", spring stiffness 300
Feel:     Professional centered dialog
```

### Alert Component
```
Appear:   Slide from right (x: 100→0) + fade in
Dismiss:  Slide right + fade out (exit mirror)
Physics:  Spring stiffness 300, damping 30, mass 0.5
Feel:     Non-intrusive notification
```

### Input Component
```
Focus:    Lift (y: -2px) + icon color change + scale 1.1
Blur:     Smooth reverse animation
Physics:  Spring stiffness 400, damping 25
Feel:     Clear visual feedback
```

### Navigation Component
```
Mobile:   Active tab highlight bar animates + icons scale
Desktop:  Background highlight animates + items slide right
Theme:    Moon/Sun icon rotates 180°
Physics:  layoutId for smooth transitions
Feel:     Polished navigation experience
```

### Page Transitions
```
Change:   Old page fade out (y: 10px) → New page fade in
```
