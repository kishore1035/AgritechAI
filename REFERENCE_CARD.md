## 🎮 Parameter Guide

| Parameter | Range | Default | Effect |

### Soil Types Explained
- 🌱 **Loam** (Best) - Balanced, most crops
- 🪨 **Clay** (Heavy) - Drainage issues
- 🏜️ **Sandy** (Light) - Water retention issues
