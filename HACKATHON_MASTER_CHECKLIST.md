## 🆘 Troubleshooting Quick Fixes

| Issue | Fix |
|-------|-----|
| **Port 5000 in use** | `Get-Process node -ErrorAction SilentlyContinue \| Stop-Process -Force` |
| **Port 5001 in use** | `Get-Process python -ErrorAction SilentlyContinue \| Stop-Process -Force` |
| **Frontend not loading** | `npm run dev` in frontend folder |
| **Login not working** | Check backend logs, ensure port 5000 is open |
| **FarmPulse not showing** | Check http://localhost:5173/farm-pulse directly |
| **White screen** | Clear browser cache (Ctrl+Shift+Delete) |
| **API errors** | Check console (F12) for specific error message |
| **Slow animations** | Disable browser extensions, try incognito mode |
